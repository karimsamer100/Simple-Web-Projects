# iOS-Style Calculator

A modern, responsive calculator web application with iOS-inspired design. Built with vanilla JavaScript, HTML5, and CSS3.

![iOS Calculator Preview](images/calculator-preview.png)

## Features

- Clean, iOS-inspired design
- Responsive layout that works on all devices
- Basic arithmetic operations (+, -, ×, ÷)
- Percentage calculations
- Positive/negative number toggle
- Thousand separators for large numbers
- Scientific notation for very large numbers
- Keyboard support with intuitive mappings
- Smooth animations and visual feedback
- Error handling for division by zero
- Maximum digit limit to prevent overflow

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- Vanilla JavaScript
- No external dependencies

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/karimsamer100/Simple-Web-Projects.git
   ```

2. Navigate to the calculator project:
   ```bash
   cd simple-web-projects/ios-calculator
   ```

3. Open `index.html` in your browser

## Usage

- Use mouse/touch to click buttons
- Keyboard shortcuts:
  - Numbers (0-9)
  - Operators:
    - Addition (+)
    - Subtraction (-)
    - Multiplication (*)
    - Division (/)
  - Equals (Enter or =)
  - Clear (Escape)
  - Decimal point (.)
  - Backspace to delete last digit
- Special Functions:
  - AC: Clear all calculations
  - ±: Toggle between positive/negative
  - %: Convert to percentage
